# Temp_Monitor

An IoT-based temperature monitoring project.

## Overview
Temp_Monitor is an embedded IoT application that reads temperature data from a BMP280 sensor and transmits it to users in real time through a gateway service. 
## Technologies
- ESP32, BMP280
- Python / C
- Wi-Fi

## Features
- Real-time temperature sensing  
- Sensor emulation and gateway integration   
- System performance monitoring
